# v1.0.1 UI 密度与倍率修复

## 目标

- 系统设置与按键映射使用一致的视觉行距。
- 固定选项组合框按最长选项内容决定宽度，减少无意义空白。
- 热键录入期间实时显示当前按下的修饰键和完整组合。
- 界面倍率同时改变字体、输入框、组合框、按钮和固定尺寸控件。
- 在倍率输入框按回车不得触发“导出方案”。
- 产品、构建目录和 Windows 版本资源升级为 v1.0.1。

## 验收

- 系统设置换行间距、单个映射内部换行间距及映射之间间距一致。
- “运行状态提示”“本步骤重复”等固定选项框宽度接近内容尺寸且不截字。
- Ctrl、Alt、Shift 按下时录键弹窗逐步显示当前组合。
- 100% 切换到 150% 后，普通文字和输入框高度均有明显增长；切回后恢复。
- 倍率输入框按 Enter 后倍率生效且导出按钮点击次数为 0。
- 完整测试、Ruff、性能/耐久与打包 EXE 冒烟通过。
