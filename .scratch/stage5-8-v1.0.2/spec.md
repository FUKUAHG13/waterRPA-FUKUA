# v1.0.2 阶段路线续作

## 目标

- 以可回退的 UI Automation 完成后台目标控制。
- 为脚本加入断点、暂停、逐步调试和受限运行表达式。
- 增加不联网的发布物料清单与完整性审核。
- 以 API 10600 能力协商和运行级熔断收束可选原生识别层。
- 只构建 Windows x64 完整多文件版。

## 不变量

- 后台操作失败不得回退真实鼠标。
- COM 控件对象不得跨线程传递，UIA 枚举必须有节点、深度和时间上限。
- 表达式禁止 `eval`、属性访问、函数调用、导入和文件/网络访问。
- 未签名发布必须明确标记，不能用测试证书伪装作者身份。
- UIA 超时且动作结果未知时不得再发送第二次点击。
- 原生 DLL 失败不得改变 Python/OpenCV 回退的脚本语义。
