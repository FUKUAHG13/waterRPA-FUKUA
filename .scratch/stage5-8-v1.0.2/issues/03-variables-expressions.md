# 运行变量与安全表达式

## Status

Completed

## Comments

- 新增设置变量和判断表达式两个稳定命令编号，真/假分别进入现有成功/失败分支。
- 静态校验与运行时均使用 AST 白名单，不调用 `eval` 或 `exec`。
- 断点变量快照只发往当前 UI，不写入运行轨迹和诊断导出。

## Acceptance

- “设置变量”支持单个名称赋值，“判断表达式”真走成功分支、假走失败分支。
- 变量只存在于一次运行，提供 loop/step/attempt/elapsed/last_success 等只读变量。
- 禁止调用、属性、下标、集合、导入和任意代码执行；长度、节点、数字、文本和变量数均有上限。
- 校验、执行、流程图、配置保存、复制粘贴和全量导出保持兼容。
