# 阶段四：可测量性能优化

## 目标

- 对截图、裁剪、模板预处理、DLL/OpenCV 匹配、点击、状态和日志刷新做有界计时。
- 消除 DLL 多缩放重复截图/积分图、Python 基础模板重复转换和同条件区域重复截图。
- 建立 1080p/2K/4K、灰度/彩色、多区域、多缩放基准与真实屏幕正向烟雾测试。
- 截图后端失败后本轮熔断，DLL 确定性预算回退本轮缓存，保持 OpenCV 回退语义。

## 当前结果

- DLL API 10500；每个区域每次调用只截图并构建一次积分图。
- 原生模板使用 64 项/256 MB 有界 LRU；Python 性能样本每项最多 256 个。
- 同机九缩放基准：当前 DLL P50 约 16.3 ms，旧版 API 10400 约 149.4 ms。
- 真实屏幕正向模板命中坐标误差 0 px。
- DXGI/Desktop Duplication 仍未接入；当前原生 GDI 失败会回退 Python MSS，MSS 失败再回退 pyautogui。

## 发布门槛

- 完整测试、Ruff F/E9、核心/视觉基准、20,000 轮耐久、原生正向烟雾和打包自检全部通过。
