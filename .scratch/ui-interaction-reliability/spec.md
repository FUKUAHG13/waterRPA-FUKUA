# UI 交互可靠性

## 目标

- 设置页同一视觉行内的标签、复选框、输入框、按钮和帮助按钮垂直居中。
- 后台窗口点击改为用户主动点选目标程序窗口，不再依据映射坐标猜测窗口。
- 绑定窗口只在“后台窗口点击(实验)”模式下可用，后台失败时不得回退真实鼠标点击。
- 热键录入稳定支持 Alt、Ctrl、Shift 及其任意组合加普通键。
- 更新长期阶段路线并构建完整多文件版。

## 验收

- 响应式设置行在宽、窄窗口和不同界面倍率下保持行内垂直对齐。
- 取点仅修改点击坐标；“选择窗口”随后要求用户到目标程序上单击一次。
- 非后台模式下选窗按钮不可用，切换模式后立即更新。
- Ctrl+A、Alt+A、Ctrl+Shift+A、Ctrl+Alt+Shift+A 均能被底层捕获并规范化。
- 全量测试、Ruff、离线构建验证通过。
