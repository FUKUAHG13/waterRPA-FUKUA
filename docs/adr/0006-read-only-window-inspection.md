# ADR 0006：只读窗口检查与权限诊断

- 状态：已接受
- 日期：2026-07-12

## 决策

后台映射绑定后允许检查根窗口、实际目标控件、客户区、进程、最多 128 个子控件，以及本程序
和目标进程的完整性级别。检查器只读取并分类为较可能有效、实验性、自绘兼容性较低或可能
被权限阻止，不自动提权，也不改变现有 PostMessage 点击方式。

## 原因

PostMessage 返回成功只代表消息进入队列，不代表游戏、浏览器画布或高权限程序会响应。
在执行前显示句柄、控件类和权限差异，比把不兼容目标伪装成成功更容易排查。

## 后果

目标权限高于 fukuaRPA 时提示用户可自行尝试管理员运行；后台失败仍不得回退真实鼠标点击。
UI Automation Invoke/Selection 属于后续独立兼容层，接入前必须保持当前绑定和回退契约。
