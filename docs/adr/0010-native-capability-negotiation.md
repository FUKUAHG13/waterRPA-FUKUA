# ADR 0010：原生能力协商与运行级熔断

- 状态：已接受
- 日期：2026-07-12

## 决策

DLL API 10600 通过 `wrpa_capabilities()` 返回稳定的 64 位能力掩码。Python 记录 API、能力、
最近返回码与错误，并始终保留 OpenCV 路径。返回空列表是权威未命中；返回负码是失败。
工作预算返回固定 `-2`，按识别签名缓存；其他失败连续三次后只在本次运行熔断原生路径。

## 原因

只比较单个版本号无法表达可选能力，依赖错误文字分类又会被翻译或提示修改破坏。运行级熔断
可以阻止损坏 DLL 在高频循环中反复失败，同时让下一次脚本运行有机会恢复。

## 后果

原生调度器不会在本阶段接管 Python 的步骤语义。增加新能力必须分配新位并保持旧位含义；
发布记录、离线自检、基准和回归测试必须验证本次发布所需能力。
