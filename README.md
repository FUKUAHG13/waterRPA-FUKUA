# waterRPA-FUKUA
对b站up不高兴就喝水的开源软件waterRPA进行了一些修改，增加了一些我认为用得上的功能
================================================================
By:b站浮夸HG

本项目基于原开源 Python 自动化脚本进行二次开发，围绕图像识别点击、脚本编排、运行控制和用户配置体验做了大量增强，目标是让非编程用户也能通过图形界面快速制作自己的自动化操作流程。

## 核心功能

- 图形化脚本编辑：支持左键单击、左键双击、右键单击、鼠标悬停、拖拽、等待、滚轮、键盘按键、弹窗提醒、声音提示、截图保存等步骤。
- 图片识别点击：可通过目标图片自动定位并执行点击，支持全局识别参数和单步骤独立识别参数。
- 多目标点击：当屏幕上出现多个相同目标时，可选择点击最佳目标或全部匹配目标，并支持从上到下、从左到右、从右到左、距离鼠标最近、随机顺序等点击策略。
- 同点位点击限制：可设置同一个图片识别点位最多点击多少次，达到上限后自动忽略该点并尝试点击下一个目标。
- 条件分支控制：每个步骤可设置成功后跳过、失败后跳过、成功后跳至指定步骤、失败后跳至指定步骤，便于制作更复杂的自动化流程。
- 单步骤重复执行：每个步骤可单独设置执行一次、指定次数或无限重复，并可配置连续失败多少次后放弃本步骤。
- 坐标快速选取：点击“取”即可直接选取屏幕坐标或拖拽轨迹，无需隐藏主界面或使用额外快捷键。
- 识别区域限制：支持框选单个识别区域，也支持高级多区域识别，只在指定区域内找图，降低 CPU 压力并提升识别速度。
- 独立设置窗口：全局设置和步骤设置均使用独立窗口，并会记住上次关闭时的位置和大小。
- 运行状态提示：脚本运行时可在屏幕角落显示当前运行状态、循环次数、步骤进度和已运行时间。
- 配置方案管理：支持新建、重命名、删除、导入、导出方案，并可打开配置目录。
- 安全停止机制：支持快捷键停止、ESC/鼠标中键急停、右上角急停、任务管理器前台急停等保护方式。
- 打包版本齐全：提供完整版和精简版，并分别支持单文件与多文件打包。

## 版本说明

### 完整版

完整版包含 OpenCV / NumPy 图像识别能力，支持相似度匹配、灰度匹配、缩放识别、多目标识别等高级功能，适合对识别稳定性和复杂场景适配要求较高的用户。

### 精简版

精简版移除了缩放识别、灰度识别、相似度/模糊匹配以及相关大型运行库，体积更小，启动和分发更轻便。精简版仅支持原图、原尺寸、颜色一致的精确图片匹配，适合固定分辨率、固定界面、目标图变化较少的场景。
