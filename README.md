# waterRPA-FUKUA
基于b站up不高兴就喝水的开源软件waterRPA进行了一些修改，增加了大量可自定义功能
================================================================
By:b站浮夸HG

本项目基于原开源 Python 自动化脚本进行二次开发，围绕图像识别点击、脚本编排、运行控制和用户配置体验做了大量增强，目标是让非编程用户也能通过图形界面快速制作自己的自动化操作流程。

## 核心功能

- 图像识别点击：支持通过目标图片在屏幕中自动识别位置，并执行左键单击、左键双击、右键单击、鼠标悬停等操作。
- 坐标直达操作：除图片识别外，也支持直接输入屏幕坐标执行点击、悬停、拖拽等操作，适合固定位置的自动化流程。
- 多目标点击策略：当屏幕中同时出现多个匹配目标时，可选择按识别顺序、距离鼠标最近、随机、从上到下、从左到右等方式执行点击。
- 同点点击上限：可设置同一个图片匹配点最多点击指定次数，达到上限后自动忽略该点位并寻找下一个目标。
- 条件分支控制：每个步骤可设置成功后跳过、失败后跳过、成功后跳至指定步骤、失败后跳至指定步骤，便于制作更复杂的自动化流程。
- 步骤独立设置：每个步骤都可以单独设置识别参数、重复次数、失败处理方式、禁止跳过、同点点击上限等选项。
- 超时处理机制：支持普通超时失败/跳过，也支持超时急停，用于在关键步骤长时间未出现时立即停止全部循环。
- 循环与起始步骤控制：支持设置脚本循环次数，也可指定从第几步开始执行，方便调试或从中途继续运行。
- 识别区域限制：支持框选一个或多个识别区域，只在指定区域内找图，降低 CPU 压力并提升识别速度。
- 性能控制选项：可调节识别频率、鼠标移动速度、点击保持时间、双击间隔等参数，兼顾执行速度与稳定性。
- 操作录制：支持录制鼠标和键盘操作，并自动生成可编辑的步骤列表，减少手动配置成本。
- 快捷键控制：支持全局快捷键启动/停止脚本，并支持复制、粘贴、撤销、重做、插入等常用编辑快捷键。
- 运行状态提示：脚本运行时可在屏幕角落显示当前运行状态、循环次数、步骤进度和已运行时间。
- 独立设置窗口：全局设置和步骤设置均使用独立窗口，并会记住上次关闭时的位置和大小。
- 多方案管理：支持创建、切换、重命名、删除不同自动化方案，并支持导入/导出配置文件，方便备份和分享。

## 版本说明

### 完整版

完整版包含 OpenCV / NumPy 图像识别能力，支持相似度匹配、灰度匹配、缩放识别、多目标识别等高级功能，适合对识别稳定性和复杂场景适配要求较高的用户。

### 精简版

精简版移除了缩放识别、灰度识别、相似度/模糊匹配以及相关大型运行库，体积更小，启动和分发更轻便。精简版仅支持原图、原尺寸、颜色一致的精确图片匹配，适合固定分辨率、固定界面、目标图变化较少的场景。
