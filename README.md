# waterRPA-FUKUA
基于b站up不高兴就喝水的开源软件waterRPA进行了一些修改，增加了大量可自定义功能
================================================================
By:b站浮夸HG

本项目基于原开源 Python 自动化脚本进行二次开发，围绕图像识别点击、脚本编排、运行控制和用户配置体验做了大量增强，目标是让非编程用户也能通过图形界面快速制作自己的自动化操作流程。


## 核心功能

- 图像识别自动点击：支持通过目标图片在屏幕中自动定位目标，并执行左键单击、左键双击、右键单击、左键/右键拖拽、鼠标悬停等操作。
- 图片内指定位置点击：识别整张目标图片后，可指定点击图片内部的某个相对位置。例如识别整只地鼠，但只点击鼻子、按钮角落或图标中心，降低小图误识别风险。
- 坐标直达与步进点击：支持直接输入屏幕坐标执行点击、悬停、拖拽等操作；也支持按固定方向或移动到目标点位进行步进点击，适合批量点击一列、一排或规则排列的目标。
- 可视化点位预览：可预览坐标点击、步进点击以及脚本中将要点击的位置。步进点位支持拖动，便于检查和对复杂脚本进行修改。
- 多目标点击策略：当屏幕中同时出现多个匹配目标时，可选择识别顺序、距离鼠标最近、随机、从上到下、从左到右等策略，也可限制同一个点位最多点击次数，避免反复点同一处。
- 高级识别区域：支持框选一个或多个识别区域，只在指定范围内找图，减少无效扫描区域，降低 CPU 压力并提升识别速度。
- 步骤级独立设置：每个步骤都可以单独设置识别参数、点击限制、重复次数、失败处理、禁止跳过、超时策略、循环生效范围等，不同步骤可以使用不同规则。
- 条件分支与跳转：支持成功后跳过、失败后跳过、成功后跳至指定步骤、失败后跳至指定步骤，可制作带判断逻辑的自动化流程，而不只是从上到下机械执行。
- 超时与急停机制：普通超时可用于当前步骤失败或跳过；超时急停可用于关键步骤长时间未出现时立即停止整个脚本，避免后续流程误操作。
- 循环控制能力：支持设置脚本循环次数、从第几步开始执行、从第几轮开始执行、到第几轮停止，也支持单个步骤只在指定循环范围内执行。
- 运行状态可视化：脚本执行时可显示当前运行状态、循环次数、步骤进度、已运行时间，并可在点击时显示点位提示，方便判断脚本是否正在运行以及刚刚点了哪里。
- 操作录制与快捷编辑：支持录制鼠标和键盘操作并生成步骤列表；支持复制、粘贴、撤销、重做、插入、拖动排序等编辑方式，降低脚本制作成本。
- 性能与稳定性控制：可调节识别频率、执行间隔、鼠标移动速度、点击保持时间、双击间隔等参数；支持自适应退避，避免长期高频识别导致 CPU 负载过高。
- 原生识别核心加速：完整版可启用原生 DLL 识别核心，用于提升截图和模板匹配性能；如果 DLL 不可用，会自动回退到 Python/OpenCV 路径，尽量兼顾速度与兼容性。
- 多方案管理：支持创建、切换、重命名、删除不同自动化方案，并支持导入/导出配置文件，方便备份、迁移和分享。
- 独立设置窗口：全局设置和步骤设置均使用独立窗口，并会记住上次关闭时的位置和大小，适合长期配置和复杂脚本维护。

## 版本说明

### 完整版

完整版包含完整的图像识别能力，适合大多数用户使用。

完整版支持 OpenCV / NumPy 识别能力，包括相似度匹配、灰度识别、缩放识别、多目标识别、多区域识别、图片内指定位置点击等功能，并可选择启用原生 DLL 识别核心以提升性能。

如果你的脚本需要适配不同分辨率、不同缩放比例、目标颜色变化、多个目标同时出现，或者需要更稳定的复杂流程控制，建议使用完整版。

### 精简版

精简版面向固定环境下的轻量使用场景。

精简版会移除缩放识别、灰度识别、模糊/相似度匹配以及相关大型运行库，体积更小，分发更轻便，但识别能力也会相应减少。

精简版更适合固定分辨率、固定窗口位置、固定界面颜色，并且目标图片变化较少的简单自动化流程。
