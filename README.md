# waterRPA-FUKUA
对不高兴就喝水大大的开源软件waterRPA进行了一些修改，增加了一些我认为用得上的功能
================================================================
不高兴就喝水 RPA配置工具 - 更新日志
版本: 浮夸改 V1.0 
By:浮夸HG

[ 核心引擎与算法升级 ]

智能多级识别策略

原版: 仅支持单一的固定相似度匹配，环境稍有变化即失效。

新版: 引入三级流水线：
A. 极速模糊匹配 (优先尝试带颜色的原图)
B. 灰度降级匹配 (自动忽略色差)
C. 智能缩放匹配 (自动识别忽大忽小的目标，支持 0.5倍 - 2.0倍)

疯狗极速内核 (Berserker Mode)

原版: 每次找图都重新读取硬盘，速度慢，点击有延迟。

新版:
A. 内存级预加载: 启动时将所有图片加载进内存，运行时零硬盘读写。
B. 单帧复用技术: 一次截屏供所有目标共用，CPU 负担大幅降低。
C. 降维打击算法: 缩放搜索时自动降低分辨率粗筛，性能提升 20 倍。

游戏级拟人点击

原版: 简单的点击指令，容易被游戏忽略。

新版: 模拟真实手指行为：按下 -> 保持(可设0.04s) -> 抬起，完美适配 Flash 游戏。

[ 新增核心功能 ]

主动避让系统 (防遮挡)

痛点解决: 防止鼠标停在目标上挡住视线导致无法识别。

功能: 点击完成后，鼠标瞬间移动到指定坐标。支持“二段闪现”，强迫游戏刷新鼠标位置。

鹰眼区域锁定

痛点解决: 全屏找图太慢且容易误触。

功能: 新增“设定识别区域”按钮，半透明遮罩框选游戏窗口。仅在框选区域内搜索，速度提升 10 倍。

系统级安全急停 (看门狗)

原版: 难以停止。

新版: 引入独立看门狗线程，20ms 高频监控：
A. 物理按键: 按下 ESC 或 鼠标中键 立即停止。
B. 鼠标手势: 甩动鼠标至屏幕 右上角 立即停止。
C. 环境感知: 检测到“任务管理器”在前台时自动停止。

[ 界面与交互体验 ]

真·拖拽排序: 任务列表支持直接用鼠标拖动行来调整顺序。

全局热键: 支持 F1-F12 自定义热键，游戏中一键启动/停止。

保姆级帮助: 关键参数后增加 [?] 按钮，点击显示详细解释。

配置记忆: 所有的参数（缩放、速度、热键、坐标等）自动保存，下次打开即用。

实时监控: 底部状态栏显示当前 CPU 逻辑核心编号及单核占用率。

[ 底层修复 ]

中文路径支持: 修复了无法识别带有中文路径图片的问题。

高分屏适配: 修复了在 2K/4K 屏幕（125%/150%缩放）下，点击位置偏移和划选区域错位的问题。

打包兼容性: 修复了 NumPy 版本冲突导致的报错，移除对特定环境的强依赖。

交互死锁修复: 修复了划定区域后窗口不消失的问题。
