# 变更记录

这里仅保留用户可感知的功能、修复和重要兼容性变化。内部实验、临时基准、开发流程和个人维护取舍不进入本文件。

## v1.0.12 - 2026-07-13

- 简易设置模式进一步收起多目标、凭据、避让、循环边界、变量表达式和窗口控件等高级功能；已有高级步骤保持原样，不会因切换视图被改写。
- “点位序列”更名为“自定义点位”，并整理界面说明、项目首页、仓库地址和源码构建文档。

## v1.0.11 - 2026-07-13

- 步骤引用改用稳定 ID，拖动、插入、复制和删除后仍能保持正确跳转；恢复拖动插入位置预览。
- 录制支持双击、组合键、滚轮合并和真实拖拽时长；新增窗口等待、激活、关闭以及 UI Automation 控件操作。
- 新增本地 DPAPI 凭据库和秘密文本输入；普通与秘密文本均不进入日志，完整导出遇到缺失图片时会明确停止。

## v1.0.10 - 2026-07-13

- 方案迁移前验证原始签名并保留备份；过新方案和过新备份进入只读保护，避免旧程序覆盖。
- 日志内容菜单支持连续勾选；修复 180% 界面倍率下主工具栏文字裁切。
- 精简冻结运行载荷，缩小包体积，并补充真实输入、截图后端和离线启动检查。

## v1.0.9 - 2026-07-13

- 日志增加可保存的自定义内容模式，统一简易、详细、完全三个预设与严重错误保留规则。
- 详细和完全日志记录本地时间与阶段耗时；资源监测可调整刷新间隔并区分系统与本进程占用。

## v1.0.8 - 2026-07-13

- 修复同一图片因缩放范围变化产生重复学习记录的问题。
- 新增画面变化唤醒：空闲等待时检测目标区域变化，优先尝试常用倍率，未命中仍立即执行完整搜索。

## v1.0.7 - 2026-07-12

- 设置页首次打开时提示快速响应场景可调整步骤间隔、识别频率和自适应降频。
- 避让坐标支持直接取点，并可选择仅移动、在最终位置左键单击或右键单击。

## v1.0.6 - 2026-07-12

- 缩放识别从记住单个倍率升级为会话内多倍率学习，支持手动优先倍率和可选容量策略。
- 启动过程拆分为轻量首帧、工作区加载和运行后端初始化，降低双击后等待主界面的感受。
- 修复原生多核选择框、连续删除悬停状态和部分界面生命周期问题。

## v1.0.5 - 2026-07-12

- 原生图片识别改用一次调用内的有界任务池处理倍率、区域和结果行块，减少重复创建线程。
- 新增原生多核自动、单线程和强制模式；快速一个优先尝试常用倍率，失败后继续完整搜索。

## v1.0.4 - 2026-07-12

- 统一设置行间距、控件垂直中心、响应式换行和滚动范围，修复紧凑按钮裁字与映射状态显示。
- 未聚焦的下拉框不再响应滚轮，连续删除步骤时保持一致的危险悬停状态。
- 条件断点移入高级调试并增加常用暂停条件预设。

## v1.0.3 - 2026-07-12

- 限制 OpenCV 工作线程并统一诊断报告，减少高核心数电脑上的无效 CPU 消耗。
- UI Automation 优先匹配明确绑定的子窗口，并按控件层级、面积和动作能力选择目标。
- 发布物料增加源码指纹、依赖清单、关键文件哈希、真实签名状态、便携 ZIP 和依赖闭包检查。

## v1.0.2 - 2026-07-12

- 后台左键操作支持 UI Automation Invoke、Selection、Toggle 与 DefaultAction，并为超时请求增加取消保护。
- 新增步骤断点、暂停、继续、单步、运行变量和受限表达式，不执行任意 Python 代码。
- 增加 CycloneDX SBOM、第三方许可证汇总、包内外完整性清单和设置页后台校验。

## v1.0.1 - 2026-07-12

- 统一设置布局和界面倍率行为，修复倍率输入按回车误触发导出与热键录入反馈问题。
- 新增单步执行、脚本流程检查、离线诊断、运行轨迹和后台目标窗口检查器。
- 加强配置生命周期、原生识别回退、启动速度和发布前性能/耐久验证基础。
